## Workflow

If you ever swap a chime:

1. Browse BigSoundBank ([bigsoundbank.com](https://bigsoundbank.com)) or Freesound.org (CC0-only filter).
2. BigSoundBank direct-download pattern: `https://bigsoundbank.com/UPLOAD/mp3/{ID}.mp3` where `{ID}` is the 4-digit sound ID from the page URL.
3. Save as `static/sounds/chime-N.mp3` (overwriting the slot).
4. Update the row above with the source URL, license, and author.
5. Audition end-to-end on a real device, in app: Settings, Töne, tap each chime preview. Reject anything that blends into ambient kitchen noise.
